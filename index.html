点滴

发表于 2017-12-11 | 阅读次数:

冬天快到了，夏天还会远吗。

在每年的冬天，就想着夏天怎么玩怎么玩，海边沙滩泳池。到了夏天，就想着冬天怎么好怎么好，一觉睡天亮，早上还可以钻个暖暖的被窝。

无论在学校，在家里，还是在上班，总有一堆心事环绕在脑子。以前总想着，在过几年吧，会越来越好的。今年虽然少了就业的压力，但是多了其他的压力，以前不会在意自己的年龄，觉得时间太多，每年都想快点到过年。不知不觉已经过了18、19岁了，光阴似箭，倏然而逝，一下子就蹦到了25了。

懵懂的青春，与蜕变

每个人的青春都跟单纯的像一张白纸，经过岁月的洗涤，再也回不到那种单纯，无论是做什么的事或者对怎么样的人都会留一点心眼。

大学之前如果喜欢一个女生，没有勇气靠近。总是小心翼翼的跟身边的朋友说，哇，我发现一个很漂亮的女生，她经常从哪里哪里经过……。那种单纯的喜欢，觉得每天能遇见一两次就很开心。

大二在考驾照时认识了一个女生，同一个教练教，所以有很多交流的机会。第一次那么主动跟别人要微信QQ，慢慢的开始聊天越来越多，有时一起跑步，也一起去过几次图书馆，认识了大半年，终于鼓起勇气表白，却得到一个很受打击的答案。失眠了一个星期，好几次半夜惊醒，好像这个世界失去了颜色一样，觉得这辈子再也不能遇到更好的了。

随着时间的流逝，这件事在脑海里越来越淡了。从懵懂的少年变的沉稳许多。从完全不了解女生心思，到渐渐清楚女生的想法。

假如时间，停止了转…

“明天应该怎么计划…，调个⏰8点起床，见到了该怎么打招呼…，中午吃什么好，下午应该去哪里好…”。第一次约女生，前一天的晚上显得有点惊慌失措，这些事仿佛就发生在昨天，有时候突然会感叹“时间啊，怎么过的这么快！”，好想认真的待好每个人！认真活好每一天！

还以为会等上半个钟，没想到，刚一坐下就看一位迷失在人群中好像在找妈妈的少女。牛仔裤、小白鞋配一件披风，嗯，还是挺会搭配的。第一眼见到好有感觉，比我想象中要淡定和有气质。

走在一起是那么的自然和舒坦，第一次希望时间慢点走……

世界上最可怕的东西莫过于心灵的孤独，灵魂的失落。

我不是个很拖的人，尤其是感情方面。特别希望有个懂自己的人，爱情可以为生活添加很多正能量。人生得一知己足矣，我觉得那个知己一定会是另一半。认识一年多了，但是始终处于不温不火状态，因为有很多原因，所以刻意保持距离。

Mac上Hexo+Github 搭建属于自己的博客（下)

前言

这是系列文章，接下来要对个人博客进行润色以及发布博文，可以看前一篇了解如何部署博客到GitHub上，事不宜迟动手吧~~

主要步骤

  • 给博客配上心仪的主题
  • 给博客配置个人信息
  • 发布博客

    给博客配上心仪的主题

  • 挑选博客主题
    Hexo官方提供许多好看的主题，预览后发现有喜欢的主题可以拿来用了。

  • 启用主题
    在Bolg目录文件下终端输入$ git clone https://github.com/iissnan/hexo-theme-next.git themes/next
    这个命令的意思是把主题拷贝到Bolg下themes文件下，themes文件下是存放所有要使用的主题，最后由站点配置文件(根目录下_config.yml文件)决定使用哪个主题。

打开Bolg下_config.yml文件，修改theme: landscape为theme: next，前提是你已经下载完了next主题

  • 部署
    终端执行$ hexo generate，编译后，会出现一个public文件夹，里面将所有的md文件编译成html文件及一些css，js，图片等文件
    $ hexo server开启本地服务，可以在本地浏览器浏览博客，部署好后可以预览http://localhost:4000/
    $ hexo deploy将本地博客部署到关联的github上，查看你的用户名.github.io

  • 站点主要配置项
    打开Bolg下站点配置文件（_config.yml），修改下主要配置项
    例如：language: zh-Hans 把语言设置汉语

  • 主题主要配置项

  • 添加侧边栏社交链接
    先进行链接的设置

然后是对应的图标

设定链接的图标，对应的字段是 social_icons。其键值格式是 匹配键: Font Awesome 图标名称， 匹配键 与上一步所配置的链接的 显示文本 相同（大小写严格匹配），图标名称 是 Font Awesome 图标的名字（不必带 fa- 前缀）。 enable 选项用于控制是否显示图标，你可以设置成 false 来去掉图标。

  • 配置完后查看效果
    终端进入Bolg下，输入$ Hexo g生成静态页面,接着输入$ Hexo s开启本地服务，先在本地看看效果http://localhost:4000/

Mac上Hexo+Github 搭建属于自己的博客（上）

发表于 2017-02-14 | 阅读次数:

前言

一直想搭建属于自己的博客，博客可以记录学习历程、总结经验。趁着周末查了几篇博客，心里有个底，话不多说，开始吧~！

主要步骤

  • 配置环境
  • 本机测试Hero
  • 申请github账号并配置
  • 本地博客部署到github上
  • 给博客配置喜欢的主题

配置环境

  • 安装git，安装完后，打开iTerm(终端),输入git –version,能查看到版本号就说明安装成功了

  • 下载node.js，下载好后先放着，待会用。

  • 安装Homebrew（前提是安装了ruby，Mac下自带ruby，所以安装ruby略过），终端输入ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”。输入brew -v检查Homebrew是否安装成功

  • 安装node，打开刚刚下载的安装包并安装

终端输入brew install node安装node，安装完同样查看版本号node -v

  • 安装Hexo，万事(git & node)具备只欠东风(Hexo)，想了解Hexo的可以看看Hexo中文官方文档 Hexo英文官方文档，话不多说安装Hexo，终端输入npm install -g hexo

如果你的终端出现了上面这些，说明安装失败，我也纠结了好久，Google了一下问题，终于找到答案，终端重新输入sudo npm install -g hexo

安装成功会出现一大串这些东东。

  • 初始化Hexo，桌面新建Blog文件（以后bolg配置文件，博文都放这里），该目录下打开终端输入hexo init，然后它会自动在Blog文件夹下生成一些配置文件，这个blog文件夹就是博客的根目录。

上面显示的证明Hero初始化成功了。新建的Bolg文件夹生成出的目录结构(想了解目录结构用途可以参考Hexo中文官方文档)

本机测试Hero

在blog目录下打开终端分别输入

  • $ hexo generate或者简写$ hexo g
    生成静态文件，会在当前目录下生成一个新的叫做public的文件夹
  • $ hexo server 或者简写$ hexo s
    启动本地web服务，用于博客的预览
  • 最激动人心的时刻
    打开浏览器输入http://localhost:4000/你会看到Hexo默认主题和一篇默认博文

    总结

    这只是本机的部署，下面还要进行github配置和Hexo主题设置，因为博客是要部署到github上的。
